Setting Up Active Directory

For this project I will be setting up active directory between two virtual machines.

First start with downloading the Windows Server 2022 iso file and aWindows 10 iso file. These will be used for the Domain Controller and the Client virtual machines.

Once you have these two files downloaded, creat the two virtual machines through Oracle Virtual Box. You can name the two machins whatever you wish but for this instance they will be namded DC, domain controller, and Windows Client respectively

Now that you have both of your virtual machines created, you first want to go into settings under network for the Client and set adapter 1 to internal. Then go into settings/network for DC and changed adapter 2(not 1!) to internal. With these two aspects finished you can now launch the DC virtual machine.

During the set up choose Desktop Experience in order to have a GUI. Depending on the specs you allocated the set up may take some time. Once the set up is finished, you will have to unlock the virtual machine by going into input at the top, selecting keyboard/ctrl+alt+delete. You can now put in your password you chose during set up.

Now, before anything you also want to go into devices at the top and select Insert Guest Additions CD Image. Then open file explorer and open the AMD file of Guest Additions. This will make for a much smoother experience.
